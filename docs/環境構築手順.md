# 環境構築手順（Docker + ROCm）

## 必要環境

| 項目 | 要件 |
|------|------|
| OS | Windows 11 + WSL2（Ubuntu 22.04） |
| GPU | AMD Radeon RX 7900 XT（gfx1100） |
| メモリ | 32GB以上推奨 |
| Docker | 29.5.3以上 |
| ROCm | 7.2.4（Dockerイメージに同梱） |

## セットアップ手順

### 1. WSL2 の準備

```powershell
# Windows PowerShell（管理者）
wsl --install -d Ubuntu-22.04
```

### 2. Docker Engine のインストール（WSL2内）

Docker Desktop（Windows版）ではなく、**WSL2上にDocker Engine** を直接インストールします。

```bash
# WSL2（Ubuntu）
curl -fsSL https://get.docker.com -o get-docker.sh
sudo sh get-docker.sh
sudo usermod -aG docker $USER
# いったんWSL2からログアウトし、再度ログイン
```

### 2. プロジェクトの配置

```bash
cd /path/to/project
git clone https://github.com/tanadeyu/bld-cell-ai-TF.git
cd bld-cell-ai-TF
```

### 3. Dockerイメージのビルド

```bash
./run_jupyter.sh
```

初回起動時に自動ビルド（約10分）。2回目以降は即座にJupyterが起動します。

### 4. Jupyter Notebook にアクセス

ブラウザで http://localhost:8888 を開く。

### 5. ノートブック実行順

| 順番 | ノートブック | 内容 | 所要時間 |
|:----:|:-----------|:-----|:--------:|
| 1 | `01_環境構築とデータ準備.ipynb` | データダウンロード | 1分 |
| 2 | `02_ResNet18学習.ipynb` | ResNet18 学習 | 各3分 |
| 3 | `03_ViT学習.ipynb` | ViT-B/16 学習 | 各6〜20分 |
| 4 | `04_最終評価.ipynb` | 結果比較 | 1分 |

各ノートブック内で `USE_FINETUNING=True`（全層学習）と `False`（最終層のみ）を切り替えて2回実行すると、両方の結果が取得できます。

## 便利コマンド

### シェルアクセス（デバッグ用）

```bash
./run_shell.sh
```

### コンテナ内で直接コマンド実行

```bash
docker exec jupyter python3 -c "コード"
docker exec jupyter pip install パッケージ名
```

## 注意事項

- GPUアクセスには `/dev/dxg` が必要（WSL GPU Paravirtualization）
- 初回起動時にDockerイメージビルドが走る（以降は即起動）
- PCクラッシュ防止のため、`run_jupyter.sh` にメモリ制限（18g）と `TF_FORCE_GPU_ALLOW_GROWTH=true` を設定済み
- XLA JITはROCm 7.2 + gfx1100で使用不可。全ノートブックはXLA無効で動作する設定済み
